# IARTNET Ingestion

Laravel monorepo for the IARTNET ingestion API (master PostgreSQL, IIIF, ETL, operator Filament panel).

## Layout

| Path | Role |
|------|------|
| `iartnet/apps/api/` | Main Laravel API application |
| `iartnet/infra/` | Docker Compose and deployment assets |
| `iartnet/docs/` | Architecture and runbooks |

See `iartnet/README.md` for detailed setup.
